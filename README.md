# react-native-shine

Fast and efficient way of adding interactive effects that are run entirely on shaders using typeGPU.

## Installation


```sh
npm install react-native-shine
```


## Usage


```js
import { multiply } from 'react-native-shine';

// ...

const result = await multiply(3, 7);
```
